第三个算法：选择排序
是一种原地排序算法（不需要额外的存储空间），适用于小文件。
扩展性差

算法：
1.寻找序列中的最小值，
2.用当前位置的值交换最小值。
3.对所有元素重复上述过程，直到整个序列排序完成。
重复选择最小的元素。
